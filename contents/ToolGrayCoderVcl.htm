(C++) GrayCoderVcl

 

GrayCoder (VCL) is a tool to convert integer values to Gray codes and vice versa.

 

GrayCoder (VCL) is the predecessor of GrayCoder.

 

The source code is hosted at the GrayCoderVcl GitHub.
