How to move a window to another desktop?

 

  • Click on the window you want to move. Then hold the Windows and Alt key, while you navigate to the target desktop with the arrow keys. The window will follow
  • Start dragging (leftclick and hold the left mouse button) the window you want to move. Then use the Windows and arrow keys to navigate to the target desktop. There, release the window
