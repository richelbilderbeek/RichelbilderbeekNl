(Game) Dissolved

 

Dissolved is an education game in which the player has to do genetic reseach, made for the 2014 G2020 e-learning hackathon.

 

Dissolved was created by team sn00b, with members:

 

  • Arlette Reneman
  • Frode Lindeijer
  • Richel Bilderbeek
  • Robbert Henning

 

The source code is hosted at the Dissolved GitHub.
