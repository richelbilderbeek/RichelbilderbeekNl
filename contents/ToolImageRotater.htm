(C++) ImageRotater

 

Finally, after 17 years of trying once-in-a-while, I've managed to create a simple tool to rotate an image under any angle.

 

There are multiple editions:

  • ImageRotater
  • ImageRotater (CLX)
  • ImageRotater (VCL)

 

The source code is hosted at the ImageRotater GitHub.
