(Game) Boenken (VCL)

 

Boenken (VCL) is game that is a crossing-over between soccer and billiards and can be played with twenty players. The game's name is Dutch, translating it to English would yield something like 'Bonking' (but if I would call the game like that, my website would attract other people). 'Boenk' is the phonetic word of the sound two colliding Dutch snooker balls make.

 

Boenken (VCL) can be played with twenty players at the same time (on the same single keyboard). Who said that video games could not be cosy?

 

Since version 3.0, BoenkenVcl was renamed Boenken and ported to Qt Creator to make use of the cross-platform Qt library.

 

The source code is hosted at the Boenken GitHub.
