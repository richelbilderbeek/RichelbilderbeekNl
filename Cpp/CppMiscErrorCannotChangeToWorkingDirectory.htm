(C++) Cannot change to working directory

 

Cannot change to working directory is a misc error in Qt Creator.

 

  • View a screenshot of the error

 

 

 

 

 

Solution

 

Go to 'Projects (CTRL + 5) | Run Settings' and uncheck 'Run in terminal'.
