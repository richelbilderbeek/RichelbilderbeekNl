(C++) Application

 

An application is 'a program that does things'.

 

There are multiple types of applications:

  • Desktop application: a program that must be installed and executed on a personal computer
  • Mobile application: a program that must be installed and executed on a mobile phone
  • Web application: a program that is installed on a server and executed in a web browser
