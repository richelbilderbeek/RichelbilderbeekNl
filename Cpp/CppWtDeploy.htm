(C++) Wt deployment

 

Wt deployment is 'changing a Wt application to a webpage'.

 

There are two types of Wt deployment: local and global:

  • Local Wt deployment denotes running a Wt application in a browser on the same machine it has been developed on, without the need of internet access. Local Wt deployment is described at the Hello Wt page.
  • Global Wt deployment denotes running a Wt application from a server, so possibly everybody with internet access can view it.
