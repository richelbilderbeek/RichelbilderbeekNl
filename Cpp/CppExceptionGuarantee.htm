(C++) exception guarantee

 

There are three types of exception guarantee:

  • basic guarantee
  • strong guarantee
  • nothrow guarantee

      •  

      Always provide the basic guarantee [1]. Provide the strong guarantee unless there is a reason not to [2].

       

       

       

       

      Reference

      2. Bjarne Stroustrup. The C++ Programming Language (4th edition). 2013. ISBN: 978-0-321-56384-2. Chapter 13.7. Advice. page 387: '[10] Provide the strong guarantee unless there is a reason not to'
