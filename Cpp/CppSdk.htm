(C++) SDK

 

An SDK (an abbreviation of 'Software Development Kit', also called 'devkit') is a collection of libraries and tools (for example, an emulator) to develop for certain operating system, environment or platform.

 

 

 

 

 

Example SDK's

 

  • Android SDK
  • Maemo SDK
  • Qt SDK
  • Symbian SDK
