(C++) Installing the Android SDK under Ubuntu

 

This page describes how to install the Android SDK under Ubuntu:

 

  • Download the Android SDK files for for Linux (i386), called 'android-sdk_r08-linux_86.tgz'
  • Extract the .tgz file and start /MyFolder/android-sdk-linux_86/tools/android (Screenshot (png))
  • The program starts. Click to download all (you need)
