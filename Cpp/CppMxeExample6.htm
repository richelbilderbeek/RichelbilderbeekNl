(C++) MXE example 6: Hello Wt

 

MXE example 6: Hello Wt is an MXE example to cross-compile a Hello Wt program from GNU/Linux to Windows.

 

Note that for me, this code does not cross-compile.

 

  • Download the Qt Creator project 'CppMxeExample6' (zip)

  • Web application

  • Lubuntu 12.04 (precise)
  • Ubuntu 12.04 (precise)

IDE(s):

  • Qt Creator 2.4.1

Project type:

  • Console application

  • C++98

Compiler(s):

  • G++ 4.6.3

Libraries used:

  • STL: GNU ISO C++ Library, version 4.6.3
  • Wt: version 3.1.10

 

 

 

 

 

Qt project file: CppMxeExample6.pro

 

QT       -= core gui
TARGET = CppMxeExample6
LIBS += \
  -lwt \
  -lwthttp
CONFIG   += console
CONFIG   -= app_bundle
TEMPLATE = app
SOURCES += main.cpp

 

 

 

 

 

main.cpp

 

#include <iostream>
#include <Wt/WString>

int main()
{
  const Wt::WString s = "Hello Wt";
  std::cout << s << '\n';
}
