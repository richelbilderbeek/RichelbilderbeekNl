(C++) std::oct

 

std::oct is a stream manipulator to let a stream display numbers in octal.

 

#include <iostream>
#include <iomanip>

int main()
{
  std::cout << std::oct;
  for (int i=0; i!=100; ++i)
  {
    std::cout << i << '\n';
  }
}
