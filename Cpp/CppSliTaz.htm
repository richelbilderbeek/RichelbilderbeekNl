(C++) SliTaz

 

SliTaz in a very small linux distribution.

 

 

 

 

 

SliTaz: install packages

 

su
tazpkg list-mirror | egrep "mypackage"
tazpkg get-install mypackage-1.2.3

 

 

 

 

 

SliTaz: start openSSH

 

In case you want to access the SliTaz computer with putty.

 

su
ssh-keygen -A
/etc/init.d/openssh start
