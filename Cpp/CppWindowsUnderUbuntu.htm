(C++) Running Windows under Ubuntu

 

There are multiple ways to run Windows under Ubuntu:

 

  • Running Windows under VirtualBox under Ubuntu: need a Windows installation CD, native environment (see this screenshot (png))
    • Running Windows under Wine under Ubuntu: no need for a Windows installation CD, simulated Windows environment (so there is no screenshot to display)
