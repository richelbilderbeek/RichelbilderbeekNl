(C++) QGraphicsItem

 

QGraphicsItem is the base class for all graphical items in a QGraphicsScene.

 

Incomplete list of derived classes of QGraphicsItem:

  • QGraphicsEllipseItem: ellipse
  • QGraphicsItemGroup: a group of QGraphicsItems
  • QGraphicsLineItem: line
  • QGraphicsObject: all graphics items that require Qt signals, slots and properties
  • QGraphicsPathItem: path
  • QGraphicsPixmapItem: pixmap
  • QGraphicsPolygonItem: polygon
  • QGraphicsProxyWidget: proxy layer for embedding a QWidget
  • QGraphicsRectItem: rectangle
  • QGraphicsSimpleTextItem: simple text
  • QGraphicsSvgItem: Scalable Vector Graphics
  • QGraphicsTextItem: formatted text
  • QGraphicsWidget: widget items
