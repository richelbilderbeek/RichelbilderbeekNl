(C++) QGraphicsSceneEvent

 

QGraphicsSceneEvent is the base class for all QGraphicsView related events.

 

Incomplete list of derived classes of QGraphicsSceneEvent:

  • QGraphicsSceneContextMenuEvent: context menu event
  • QGraphicsSceneDragDropEvent: drag and drop event
  • QGraphicsSceneHelpEvent: tooltip request event
  • QGraphicsSceneHoverEvent: hover events
  • QGraphicsSceneMouseEvent: mouse events
  • QGraphicsSceneMoveEvent: widget moving event
  • QGraphicsSceneResizeEvent: widget resizing event
  • QGraphicsSceneWheelEvent: mouse wheel events
