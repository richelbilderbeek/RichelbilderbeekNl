(C++) Hello Qwt

 

Hello Qwt is an extension of Hello World. Like Hello World, Hello Qwt is a very simple application. Hello Qwt, however, also requires the Qwt libraries.

 

Here are some detailed examples of Hello Qwt, depending on IDE and operating system:

  • 'Hello Qwt' using Qt Creator under Cygwin
  • 'Hello Qwt' using Qt Creator under Lubuntu
  • 'Hello Qwt' using Qt Creator under Lubuntu, crosscompile to Windows
