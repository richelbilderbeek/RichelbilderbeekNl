(C++) Emulate

 

Emulation is a way (according to some definitions) to port your application from environment A to environment B, by copying the executable from environment A and run it with an emulator under environment B.

 

Some emulators/emulator-like programs are:

  • Darling: runs Darmin/OS X executables under Linux
  • qemu: runs many operating systems under many operating systems, for example Raspbian under Lubuntu
  • Virtualbox: runs many operating systems under many operating systems
  • WINE: runs many Windows executables under Linux
