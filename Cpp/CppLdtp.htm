(C++) ldtp

 

ldtp is a cross platform GUI test automation tool for Linux. For Windows, it is called 'Cobra' and for Mac it is called 'PyATOM'.

 

Make sure you have run the following command at least once in the command line, otherwise ldtp will not be detected in Python:

 

gsettings set org.gnome.desktop.interface toolkit-accessibility true
