(C++) 'Hello World' using Qt Creator under Wine under Ubuntu for Symbian

 

This page shows how to create a Hello World program using (a Windows version of) Qt Creator under Wine under Ubuntu for the Symbian mobile application platform.

 

Because Symbian does not support console applications, the most basic program is 'Hello Qt' using Qt Creator under Wine under Ubuntu for Symbian.
