(C++) cstdlib.h

 

cstdlib.h is a standard header file.

 

cstdlib.h denotes 'C standard library' and it is the C++ header file version of the C header file 'stdlib.h'.

 

cstdlib.h contains the definitions of the following functions (incomplete list, taken from the GNU ISO C++ library's cstdlib.h supplied with Qt Creator 1.3.1):

 

  1. std::abort
  2. std::abs
  3. std::atexit
  4. std::atof
  5. std::atoi
  6. std::atol
  7. std::bsearch
  8. std::calloc
  9. std::div
  10. std::exit
  11. std::free
  12. std::getenv
  13. std::labs
  14. std::ldiv
  15. std::malloc
  16. std::mblen
  17. std::mbstowcs
  18. std::mbtowc
  19. std::qsort
  20. std::rand
  21. std::realloc
  22. std::srand
  23. std::strtod
  24. std::strtol
  25. std::strtoul
  26. std::system
  27. std::wcstombs
  28. std::wctomb
