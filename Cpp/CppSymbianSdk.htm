(C++) Symbian SDK

 

The Symbian SDK is the SDK to develop for the Symbian platform.

 

The Symbian SDK is developed for the Windows operating system. Those using the Ubuntu operating system can use Wine to simulate a Windows environment.

 

You will also need to download the Open C/C++ Plug-in.
