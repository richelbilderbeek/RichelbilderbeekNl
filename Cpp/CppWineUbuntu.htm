(C++) Running Windows under Wine under Ubuntu

 

Running Windows under Wine under Ubuntu is one of the ways to run Windows under Ubuntu.

 

Wine is a free program that simulates a Windows environment and allows calling of Windows executables. Because Windows under Wine is non-native ('real'), not everything works.
