(C++) 'Hello Boost' using Qt Creator under Ubuntu

 

Hello Boost is an extension of Hello World. Like Hello World, Hello Boost is a simple console application. Hello Boost, however, also requires the Boost library and to link against the Boost.Regex library.

 

  • Download the Qt Creator project 'HelloBoostQtCreatorUbuntuNds' (zip)

 

 

 

 

 

Project environment

 

  • NDS
  • Ubuntu 10.04 LTS Lucid Lynx

IDE(s):

  • Qt Creator 2.0.0

Project type:

  • Console application

Compiler(s):

  • G++ 4.4.1

Libraries used:

  • Boost: version 1.40
  • libnds: version 1.4.7
  • STL: from GCC, shipped with Qt Creator 2.0.0
