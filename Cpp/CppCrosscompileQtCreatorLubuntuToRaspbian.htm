(C++) How to cross-compile a Qt Creator project from Lubuntu to Raspbian?

 

How to cross-compile a Qt Creator project from Lubuntu to Raspbian? describes hoe to cross compile a Qt Creator project from Lubuntu to Raspbian.

http://www.kitware.com/blog/home/post/426 Downloaded wget http://crosstool-ng.org/download/crosstool-ng/crosstool-ng-1.18.0.tar.bz2 tar xjf crosstool-ng-1.18.0.tar.bz2 cd crosstool-ng-1.18.0 sudo apt-get install gperf sudo apt-get install texinfo sudo apt-get install gawk sudo apt-get install libncurses5-dev ./configure --prefix=/home/richel/pi make make install export PATH="${PATH}:/home/richel/pi/bin" Now the executable ct-ng is present in /home/richel/pi http://elinux.org/RPi_Kernel_Compilation git clone git://github.com/raspberrypi/tools.git On Pi: mkdir to_copy zcat /proc/config.gz > to_copy/.config #Otherwise conflict with the existing .config folder
