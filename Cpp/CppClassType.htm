(C++) class type

 

A class type is a type of class. Examples are:

  • Abstract base class
  • Abstract class
  • Base class
  • Container
  • Derived class
  • Enum class
  • Functor
  • POD (a C-style struct)
  • Template class
  • Winnebago class: a do-it-all class
