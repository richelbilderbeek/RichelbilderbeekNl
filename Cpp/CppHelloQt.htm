(C++) Hello Qt

 

Hello Qt is an extension of Hello World. Like Hello World, Hello Qt is a very simple application. Hello Qt, however, also requires the Qt libraries.

 

Here are some detailed examples of Hello Qt, depending on IDE and operating system:

  • 'Hello Qt' using Qt Creator under Cygwin
  • 'Hello Qt' using Qt Creator under Lubuntu
  • 'Hello Qt' using (a Windows version of) Qt Creator under Wine under Ubuntu
  • 'Hello Qt' using Qt Creator under Ubuntu
  • 'Hello Qt' using Qt Creator under Ubuntu for Symbian
  • 'Hello Qt' using Qt Creator under Windows
  • 'Hello Qt' using Qt Creator under Windows under VirtualBox under Ubuntu for Symbian
