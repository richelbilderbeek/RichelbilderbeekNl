(C++) User I/O

 

User I/O (an abbreviation of 'user input/output') is about communication with the user.

 

 

 

 

 

User I/O STL functions and data types

 

  • std::cerr: error message stream
  • std::cin: user input stream
  • std::clog: log message stream
  • std::cout: screen output stream
  • std::getline: obtain a single line of input
