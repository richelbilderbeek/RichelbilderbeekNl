(C++) Flood::MultilayerPerceptron

 

Flood::MultilayerPerceptron is a Flood data type for a multilayer perceptron.

 

 

 

 

 

Flood::MultilayerPerceptron bug

 

Note: this bug is known to the author of Flood and he kindly delivered the following patch: download the patch of this bug.

 

Some simple getters seem to give errors. The error depends on the index of the hidden layer where I request the weights from. See the code below.

 

  • Download the Qt Creator project 'CppFloodBug1' (zip)

 

Operating system: Ubuntu 10.04 LTS Lucid Lynx

IDE: Qt Creator 2.0.0

Project type: GUI application

Compiler: G++ 4.4.1

Libraries used:

  • Flood: version 3

 

 

 

 

 

Qt project file

 

#-------------------------------------------------
#
# Project created by QtCreator 2010-08-15T20:28:01
#
#-------------------------------------------------
QT += core
QT -= gui
TARGET = CppFloodBug1
CONFIG += console
CONFIG -= app_bundle
TEMPLATE = app
SOURCES += main.cpp
SOURCES += ../../Flood/MultilayerPerceptron/MultilayerPerceptron.cpp
SOURCES += ../../Flood/Perceptron/Perceptron.cpp

 

 

 

 

 

main.cpp

 

#define _DEBUG
#include "../../Flood/MultilayerPerceptron/MultilayerPerceptron.h"

int main()
{
  Flood::MultilayerPerceptron t(3,3,3);
  t.get_hidden_layer_synaptic_weights(1);
}

 

Screen output depends. Sometimes the program gives an access violation. Sometimes I get:

 

Flood Error: Matrix Template.
Constructor Matrix(int, int).
