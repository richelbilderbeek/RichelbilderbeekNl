(C++) Symbian

 

Symbian is a platform for mobile applications.

 

The Qt SDK aids Symbian development in the Qt Creator IDE by adding this as a 'Project Type' and using Qt Simulator to simulate a Symbian smartphone. As far as I know, this does not enable one to create a Symbian executable.
