(C++) VirtualBox

 

VirtualBox is a program to run an operating system natively under another operating system.

 

Raspbian cannot be run in VirtualBox, use qemu instead.

 

  • Running Windows under VirtualBox under Ubuntu
  • Running raspbian under VirtualBox under Lubuntu: will not work [1]

 

 

 

 

 

References

 

  1. http://raspberrypi.stackexchange.com/a/1192 from 'Gilles': 'VirtualBox lets you run x86 virtual machines on an x86 processor. Raspbian is a distribution for ARM processors. Raspbian cannot run in VirtualBox.'
