(C++) What are the differences between C++ Builder and Qt Creator?

 

IDE C++ Builder/RAD studio Qt Creator
Owner Embarcadero Technologies Qt Development Frameworks
Platform Windows Linux, Macintosh, Windows and more
How to deploy on other platform Use emulator on executable, (cross-)compile code on other platform
Licence One month trial or commercial GPL or commercial
Default compiler BCC32.EXE G++
Boost compatibility BCC32.EXE is an unsupported compiler G++ is a supported compiler
Cpp11 adoption Pre-standard adoption Adoption after publication of official Cpp11 standard
Graphical libraries CLX (abandoned), VCL Qt
