(C++) Compiling STL code under Cygwin

 

Compiling STL code under Cygwin is expected to be the simplest example of compiling under Cygwin, under Windows. The goal of compiling under Cygwin is to port code from Linux to Windows.

 

 

 

 

 

Environment

 

Operating system: Windows XP

IDE: any text editor (but code was developed under Qt Creator 2.0.0)

Project type: console application

Compiler: G++ 4.4.1

Libraries used:

  • STL: from GCC, shipped with Qt Creator 2.0.0

Project options:

  • Unchecked 'Projects -> Build settings -> General -> Shadow build' (so code and makefile are in the same folder)
