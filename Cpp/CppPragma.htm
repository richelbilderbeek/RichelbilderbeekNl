(C++) #pragma

 

#pragma is a compiler-specific preprocessor directive.

 

The example code below produces no screen output, but gives the compiler message below.

 

 

#pragma message "Hello world"

int main()
{

}

 

Screen output:

 

 

 

Compiler output:

 

#pragma message: Hello world
